### 查询系统整体磁盘使用情况

```
df -h
```



查询指定目录的磁盘占用情况

```
du -h             默认查询当前目录
du -h /目录
```

|     符号      |            说明            |
| :-----------: | :------------------------: |
|      -s       |    指定目录占用大小汇总    |
|      -h       |         带计量单位         |
|      -a       |           含文件           |
| --max-depth=1 |  子目录深度，1就是根目录   |
|      -c       | 列出明细的同时，增加汇总值 |

```
查看/opt 深度为1的目录
du -h --max-depth=1 /opt
```
