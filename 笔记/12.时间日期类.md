### 显示当前时间

|           语法            |     解释     |
| :-----------------------: | :----------: |
|           date            | 显示当前时间 |
|        data "+%Y"         |   当前年份   |
|        data "+%m"         |   当前月份   |
|        data "+%d"         |    当前日    |
| data "+%Y-%m-%d %H:%M:%S" | 年月日时分秒 |

这里的双引号貌似可以去掉



### 修改系统日期

```
date -s "2011-11-01 20:18:21"
date -s "2011-11-01"
date -s "20:18:21"
```
