## 关机重启

|      指令       |      解释      |
| :-------------: | :------------: |
| shutdown -h now |    立即关机    |
|  shutdown -h 1  |  1分钟后关机   |
|    shutdown     |  1分钟后关机   |
|   shutdown -r   |  1分钟后重启   |
| shutdown -r now |    立即重启    |
|      halt       |    立即关机    |
|     reboot      |  立即重启重启  |
|      sync       | 内存同步到磁盘 |

注：h是halt等待的意思



## 登录注销

在开发环境中，尽量少用root账户，避免操作失误，可以利用普通用户登录，登录后再使用`su - 用户名`命令来切换成系统管理员身份



注销用`logout`即可
